# Modern Aviation — Supervisor Start-of-Shift Checklist (Daily)
**Version:** v1.0  
**Updated:** 2026-03-04

---

## Metadata

**Purpose**  
Prevent preventable escalations by confirming schedule, priorities, equipment readiness, compliance tasks, and communication coverage.

**Trigger**  
Start of shift (AM/PM) or upon taking operational control mid-shift.

**Duration**  
20–30 minutes (target)

**Owner**  
On-duty Supervisor

**Required Items**  
Phone (charged), radio, PPE (incl. cleats if freezing), access to GroupMe, keys as needed

**Inputs**  
Modern Aviation chat, MA-Supervisors chat, Altus/Modern Ops chat, X1/RTS/JFQC status, MaintainX status, hangar logs (CSR)

**Outputs**  
Shift mini-plan, acknowledged ETAs in ops chats, critical readiness checks completed, delegated task owners, escalations sent (if needed)

**Stop Conditions**  
CRITICAL safety event, aircraft emergency, management directive to pause ops, unsafe ramp conditions

---

## PREP

**Objective**  
Arrive ready to lead: tools, access, and safety posture in place.

**Checklist**

- [ ] Phone battery                     Checked
- [ ] Radio / comms                     Checked
- [ ] PPE on-hand                       Set
- [ ] Weather readiness                 Checked → Cleats/salt plan if freezing
- [ ] GroupMe access                    Confirmed
- [ ] Keys / access basics              Confirmed → Line room / shop / gates as needed

## START

**Objective**  
Build the “operational picture” fast: schedule, priorities, constraints, and compliance risk.

**Checklist**

- [ ] Chat triage (3 channels)          Complete

    - [ ] Modern Aviation chat           Checked → Early/late ops, hangar requests, fuel drops, notices
    - [ ] MA-Supervisors chat            Checked → Dwight/Ahsan priorities, audits, cleaning, training, GSE notes
    - [ ] Altus/Modern Ops chat          Checked → MX moves, engine runs, AOG, door/heat issues

- [ ] CSR schedule review               Checked → Ingrid/Rebecca/Lacian/Jah posts
- [ ] Early ops / late ops list          Noted → Tails, online/POL/DEP, “leave in hangar” flags
- [ ] Hangar assignments                 Confirmed → Space, door constraints, “Hangar C closed” notices
- [ ] Altus/MX requests flagged          Noted → Jose Castillo / Chris Scida / Leo Katsaros / Zarzano / Domenic
- [ ] AOG mentions                        Noted → Tail + location + need
- [ ] CRITICAL: Acknowledge pending ops  Sent → Copy + constraint + ETA (no silence)
- [ ] Staffing reality check             Confirmed → Who is where (North/South), breaks, meetings
- [ ] Dwight compliance scan             Noted → Audits / deice checks / cones-chocks / lounge / QC due
- [ ] Shift mini-plan (Notes)            Logged

    - [ ] Top 3 time-critical moves      Logged
    - [ ] Top 2 compliance tasks         Logged
    - [ ] Top 1 constraint/risk          Logged → Staffing/equipment/weather/doors

## CORE

**Objective**  
Verify readiness and remove the repeat-failure points: deice, fuel/QC, blocking, lounge, and fast ETAs.

**Checklist**

- [ ] Ramp walk (primary base first)     Complete

    - [ ] Safety lanes / exits            Checked → No equipment blocking doors/exits
    - [ ] Fire extinguishers              Checked → Not blocked
    - [ ] Hangar door alignment           Checked → Heat indicators aligned (Hangar C procedure if applicable)
    - [ ] FOD / slip hazards              Checked → Ice/black ice addressed

- [ ] Deice readiness                     Complete

    - [ ] Deice truck status              Checked → OOS status known and shared
    - [ ] Running fuel level              Checked
    - [ ] Type I level                    Checked → Based on paperwork (not guess)
    - [ ] Type IV level                   Checked → If applicable / tank limits noted
    - [ ] RECORD: Daily deice checklist   Logged → No sticky-note logging
    - [ ] Binder inventory numbers        Verified → Updated if changed

- [ ] Fuel/QC readiness                   Complete

    - [ ] JFQC/RTS truck status           Checked → No unintended OOS blocks QC logging
    - [ ] QC due items (today)            Noted → Mogas / Diesel / Generators / trucks / self-serve
    - [ ] Fuel farm pump status           Checked → Known working / issue noted
    - [ ] Leaks / obvious defects         Checked → 5250/521/etc. as applicable
    - [ ] RECORD: Fuel drops status       Noted → Jet A / Avgas / Mogas drops received + paperwork location

- [ ] West Side Lounge standard           Complete

    - [ ] Garbage removed                 Checked
    - [ ] Paper goods stocked             Checked → TP / paper towels
    - [ ] Soap stocked                    Checked
    - [ ] Surfaces / glass                Checked
    - [ ] Vacuum / floor                  Checked

- [ ] Cones / chocks discipline           Checked → Stored neatly / staged correctly

- [ ] Assign owners (no “spray tasks”)    Complete

    - [ ] Deice checks owner               Assigned → (e.g., Thomas Doll / Slaw / Hunter)
    - [ ] Audits owner                     Assigned → (e.g., Hunter / Slaw / Lucas)
    - [ ] Lounge owner                     Assigned → (e.g., Zack / Eric / Thomas)
    - [ ] Altus moves lead                 Assigned → Single point of control (e.g., Eric / Chris F)
    - [ ] Facilities owner                 Assigned → Harvard / Juan for snow/heat/doors

- [ ] Altus/MX comms discipline           Complete

    - [ ] Requests acknowledged fast       Sent → Copy + ETA
    - [ ] ETAs updated if slipping         Sent → Before they ping again
    - [ ] Wing-walker need flagged         Noted → Ask MX for support if short

- [ ] CSR comms discipline                Complete

    - [ ] Constraints shared               Noted → Staffing/doors/OOS/weather
    - [ ] Customer expectations confirmed  Confirmed → Fees, “must go in hangar,” “wait for crew”

- [ ] RECORD: MaintainX entries quality   Verified

    - [ ] Picture/video attached           Confirmed
    - [ ] Description clear                Confirmed
    - [ ] Unit # + location                Confirmed

## CLOSEOUT

**Objective**  
Lock in control: confirm critical tasks are tracked, escalations sent, and next handoff is clean.

**Checklist**

- [ ] Critical tasks status                Checked → Early/late ops tails on track
- [ ] AOG status check                      Checked → If open, owner + ETA still valid
- [ ] Dwight-facing items updated           Sent → Audits/deice/QC/lounge exceptions
- [ ] Ahsan-facing items updated            Sent → Staffing constraints, move backlog, escalation risks
- [ ] CSR-facing items updated              Sent → Delays, hangar constraints, OOS services, fees
- [ ] RECORD: Drop paperwork location       Noted → North/South line room placement
- [ ] Next-hour plan                         Logged → Movements window + service window

## EXCEPTIONS

<details>
<summary>AOG declared (Aircraft on Ground)</summary>

- [ ] CRITICAL: Acknowledge AOG             Sent → Tail + location + need + ETA
- [ ] Assign AOG movement lead              Assigned
- [ ] Secure aircraft (cones/chocks)        Set
- [ ] Coordinate MX/crew availability       Confirmed
- [ ] Notify Ahsan (coordination)           Sent
- [ ] Notify Dwight (if compliance/safety)  Sent
- [ ] Update ETA before asked               Sent
- [ ] RECORD: Services/times                Logged

</details>

<details>
<summary>Deice request or deice truck OOS</summary>

- [ ] CRITICAL: Confirm deice capability    Verified → Truck + fluid + trained operator
- [ ] If truck OOS, notify CSR immediately  Sent
- [ ] Notify Dwight (inventory/checks)      Sent
- [ ] RECORD: Fluid test + checklist        Logged
- [ ] If customer impact, provide ETA       Sent → Copy + constraint + ETA

</details>

<details>
<summary>Fuel farm pump failure / fueling blocked</summary>

- [ ] Confirm reset attempts                Checked → Reset/circuit as applicable
- [ ] Notify Max / maintenance              Escalated
- [ ] Notify Ahsan                           Sent
- [ ] Notify CSR (service impact)           Sent
- [ ] If drop affected, coordinate reroute  Escalated → Swissport farm if available
- [ ] RECORD: MaintainX ticket              Logged

</details>

<details>
<summary>JFQC/RTS shows truck OOS blocking QC logging</summary>

- [ ] Confirm truck status physically       Checked
- [ ] Request AM supervisor un-OOS          Escalated → RTS/JFQC update
- [ ] RECORD: QC completion note            Logged
- [ ] Notify Dwight (if audit exposure)     Sent

</details>

<details>
<summary>Snow / black ice / ramp unsafe</summary>

- [ ] CRITICAL: Cleats enforced             Confirmed
- [ ] Salt/sand priority areas              Set → FBO entrances, hangar thresholds, alleys
- [ ] Blocked doors/lanes cleared           Escalated → Juan/Harvard/snow vendor
- [ ] Notify CSR (service delays)           Sent
- [ ] RECORD: Hazard locations              Noted → Photos if helpful

</details>

<details>
<summary>Media / law enforcement calls (incident)</summary>

- [ ] HOLD: Do not share details            Confirmed
- [ ] HANDOFF: Forward to management        Sent
- [ ] RECORD: Caller info                   Logged → Name/agency/number/time

</details>
