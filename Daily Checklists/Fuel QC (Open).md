# Fuel QC — Daily Opening / Facility ISP Checklist

**Version:** v2.0  
**Updated:** 2026-03-07  
**Duration:** 00:35

---

## Metadata

**Purpose**  
Complete the daily opening fuel QC, required 24-hour sump, preliminary DP entry, and facility ISP condition audit.

**Trigger**  
Start of morning shift before truck top-off or fueling activity.

**Owner**  
Line Service / Fuel QC

**Required Items**  
White bucket, sump tools, inspection access, ISP form, JQC access, pen/device

**Inputs**  
Truck ID, tank status, facility status, daily inspection record, ATA 103 criteria

**Outputs**  
Daily opening checks completed, preliminary filter DP entered, facility ISP completed, unresolved flow fields left pending

**Stop Conditions**  
Contaminated fuel, visible water, security issue, fire deficiency, unsafe equipment condition, missing inspection access

---

## PREP

**Objective**  
Confirm the correct unit, tools, forms, and system access before beginning daily opening checks.

**Checklist**

- [ ] Truck / tank ID Confirmed
- [ ] Daily record Open
- [ ] Facility ISP form Open
- [ ] White bucket Present
- [ ] Sump tools Ready
- [ ] JQC access Verified
- [ ] Inspection route Defined
- [ ] Fuel area condition Checked

---

## DAILY FUEL QC

**Objective**  
Perform the once-per-24-hour sump and record the preliminary morning fuel-quality findings.

**Checklist**

- [ ] Sump cycle Verified → Within 24 hours

- [ ] Tank Checked
  - [ ] Tank sump Complete
  - [ ] Sample appearance Observed
  - [ ] Visible water None
  - [ ] Sediment / debris None

- [ ] Filter Checked
  - [ ] Filter sump Complete
  - [ ] White bucket sample Observed
  - [ ] Preliminary filter DP Logged
  - [ ] GPM value N/A → Morning check is past meter

- [ ] Morning-only fields Noted → Await top-off / flow event

### DECISION: Contamination or abnormal sample present?

**YES**

- Stop fueling / QC progression
- Escalate to supervisor
- RECORD: Log discrepancy
- Hold affected unit out of use as directed

**NO**

- Continue daily opening workflow

---

## FACILITY ISP

**Objective**  
Inspect the listed facility condition and safety items during the same daily opening cycle.

**Checklist**

- [ ] General condition Checked
- [ ] Security Checked
- [ ] Fire deficiencies Checked

- [ ] Hoses / reels Checked
  - [ ] Swivel hoses Checked
  - [ ] Static reels Checked

- [ ] Fire equipment Checked
  - [ ] Fire extinguishers Checked
  - [ ] Extinguisher access Clear

- [ ] Drainage / separation Checked
  - [ ] Sump separators Checked

- [ ] Visible damage None
- [ ] Immediate hazards None

### DECISION: Was any safety, fire, or security deficiency found?

**YES**

- Escalate to supervisor
- RECORD: Log discrepancy
- Hold affected area / equipment as directed

**NO**

- Continue daily opening workflow
