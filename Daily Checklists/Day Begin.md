# Modern Aviation — Start-of-Shift Checklist (Daily)

**Version:** v1.0  
**Updated:** 2026-03-04

---

## Metadata

**Purpose**  
Prevent preventable escalations by confirming schedule, priorities, equipment readiness, compliance tasks, and communication coverage.

**Trigger**  
Start of shift (AM/PM) or upon taking operational control mid-shift.

**Duration**  
20–30 minutes (target)

**Owner**  
On-duty Supervisor

**Required Items**  
Phone (charged), radio, PPE (incl. cleats if freezing), access to GroupMe, keys as needed

**Inputs**  
Modern Aviation chat, MA-Supervisors chat, Altus/Modern Ops chat, X1/RTS/JFQC status, MaintainX status, hangar logs (CSR)

**Outputs**  
Shift mini-plan, acknowledged ETAs in ops chats, critical readiness checks completed, delegated task owners, escalations sent (if needed)

**Stop Conditions**  
CRITICAL safety event, aircraft emergency, management directive to pause ops, unsafe ramp conditions

---

## PREP

**Objective**  
Arrive ready to lead: tools, access, and safety posture in place.

- [ ] Clock In → Checked
- [ ] Phone battery Checked → If Low, set low batt mode
- [ ] Radio / comms Checked
- [ ] Ear Protection → In Leg Pouch
- [ ] Backup ear Protection → In Leg Pouch
- [ ] Vest → Know it's location
- [ ] Leather Gloves → In Leg Pouch
- [ ] Chemical-Resistant Gloves → On Clip
- [ ] Weather readiness Checked → Cleats/salt plan if freezing
- [ ] Keys / access basics Confirmed → Line room / shop / gates as needed

## START

**Objective**  
Build the “operational picture” fast: schedule, priorities, constraints, and compliance risk.

### **Chat Triage**

- [ ] Modern Aviation chat Checked
  - [ ] Early/late ops posts → Online/POL/DEP times
  - [ ] Hangar Movements → Assignments + Any hangars closed? + Any hangars full? + “leave in hangar” notes
  - [ ] Fuel/QC/drop notices → Drops received + OOS GSE + additional audits due
  - [ ] Weather/ramp alerts → Black ice + cleats + salting + snow access issues

- [ ] MA-Supervisors chat Checked

- [ ] Altus/Modern Ops chat Checked
  - [ ] AOG / “ASAP” requests → Tail + what they need + when they need it
  - [ ] Engine runs / defuel / moves → Time windows + staging location
  - [ ] Follow-ups/escalations → Tag management

- [ ] Today’s schedule picture → Top 3 tails + times
- [ ] Coverage picture → Who is North/South + breaks/meetings
- [ ] CRITICAL: Pending requests reply Reply → Copy + constraint + ETA
- [ ] Shift priorities
  - [ ] Top 3 time-critical actions → Noted
  - [ ] Top 2 compliance actions → Noted
  - [ ] Main constraint/risk → Noted

## CLOSEOUT

**Objective**  
Lock in control: confirm critical tasks are tracked, escalations sent, and next handoff is clean.

**Checklist**

- [ ] Critical tasks status Checked → Early/late ops tails on track
- [ ] AOG status check Checked → If open, owner + ETA still valid
- [ ] Dwight-facing items updated Sent → Audits/deice/QC/lounge exceptions
- [ ] Ahsan-facing items updated Sent → Staffing constraints, move backlog, escalation risks
- [ ] CSR-facing items updated Sent → Delays, hangar constraints, OOS services, fees
- [ ] RECORD: Drop paperwork location Noted → North/South line room placement
- [ ] Next-hour plan Logged → Movements window + service window

## EXCEPTIONS

<details>
<summary>AOG declared (Aircraft on Ground)</summary>

- [ ] CRITICAL: Acknowledge AOG Sent → Tail + location + need + ETA
- [ ] Assign AOG movement lead Assigned
- [ ] Secure aircraft (cones/chocks) Set
- [ ] Coordinate MX/crew availability Confirmed
- [ ] Notify Ahsan (coordination) Sent
- [ ] Notify Dwight (if compliance/safety) Sent
- [ ] Update ETA before asked Sent
- [ ] RECORD: Services/times Logged

</details>

<details>
<summary>Deice request or deice truck OOS</summary>

- [ ] CRITICAL: Confirm deice capability Verified → Truck + fluid + trained operator
- [ ] If truck OOS, notify CSR immediately Sent
- [ ] Notify Dwight (inventory/checks) Sent
- [ ] RECORD: Fluid test + checklist Logged
- [ ] If customer impact, provide ETA Sent → Copy + constraint + ETA

</details>

<details>
<summary>Fuel farm pump failure / fueling blocked</summary>

- [ ] Confirm reset attempts Checked → Reset/circuit as applicable
- [ ] Notify Max / maintenance Escalated
- [ ] Notify Ahsan Sent
- [ ] Notify CSR (service impact) Sent
- [ ] If drop affected, coordinate reroute Escalated → Swissport farm if available
- [ ] RECORD: MaintainX ticket Logged

</details>

<details>
<summary>JFQC/RTS shows truck OOS blocking QC logging</summary>

- [ ] Confirm truck status physically Checked
- [ ] Request AM supervisor un-OOS Escalated → RTS/JFQC update
- [ ] RECORD: QC completion note Logged
- [ ] Notify Dwight (if audit exposure) Sent

</details>

<details>
<summary>Snow / black ice / ramp unsafe</summary>

- [ ] CRITICAL: Cleats enforced Confirmed
- [ ] Salt/sand priority areas Set → FBO entrances, hangar thresholds, alleys
- [ ] Blocked doors/lanes cleared Escalated → Juan/Harvard/snow vendor
- [ ] Notify CSR (service delays) Sent
- [ ] RECORD: Hazard locations Noted → Photos if helpful

</details>
