**Version:** v1.0  
**Updated:** 2026-03-07  
**Duration:** 00:20

---

## Metadata

**Purpose**  
Complete the daily 24-hour sump and capture preliminary morning fuel-quality checks.

**Trigger**  
Start of morning shift before truck top-off or fueling activity.

**Owner**  
Line Service / Fuel QC

**Required Items**  
White bucket, sump tools, inspection access, JQC access, pen/device

**Inputs**  
Truck ID, tank status, daily inspection record, ATA 103 criteria

**Outputs**  
Morning sump completed, preliminary filter DP entered, unresolved fields left pending

**Stop Conditions**  
Contaminated fuel, visible water, missing inspection access, unsafe equipment condition

## CORE

**Objective**  
Perform the once-per-24-hour sump and record the preliminary morning findings.

**Checklist**

- [ ] Sump frequency                    Verified → Within 24 hours
- [ ] Tank sump                         Complete
- [ ] Filter sump                       Complete
- [ ] White bucket sample               Observed
- [ ] Visible water                     None
- [ ] Sediment / debris                 None
- [ ] Fuel appearance                   Checked
- [ ] Preliminary filter DP             Logged
- [ ] GPM value                         N/A → Morning check is past meter
- [ ] Morning-only fields               Noted → Await top-off / flow event

### DECISION: Contamination or abnormal sample present?

**YES**
- Stop fueling / QC progression
- Escalate to supervisor
- RECORD: Log discrepancy
- Hold affected unit out of use as directed

**NO**
- Continue daily inspection workflow
