# Fuel QC — MX Dashboard / Calibration & Recurring Maintenance Tracking Checklist

**Version:** v1.0  
**Updated:** 2026-03-07  
**Duration:** 00:20

---

## Metadata

**Purpose**  
Track and close non-daily, non-weekly QC items on the MX dashboard.

**Trigger**  
Scheduled maintenance / calibration due date or dashboard review cycle.

**Owner**  
Fuel QC / Supervisor

**Required Items**  
MX dashboard access, calibration records, unit list, maintenance support as needed

**Inputs**  
Open MX items, due dates, calibration results, truck applicability

**Outputs**  
Dashboard updated, completed items submitted, discrepancies escalated

**Stop Conditions**  
Failed calibration, missing maintenance record, unresolved out-of-service condition

---

## PREP

**Objective**  
Review all open MX items and sort by type and due date.

**Checklist**

- [ ] MX dashboard Open
- [ ] Open items Reviewed
- [ ] Due dates Checked
- [ ] Unit applicability Reviewed

**Checklist**

- [ ] Quarterly items Reviewed
- [ ] Annual items Reviewed

- [ ] DP calibrations Reviewed
- [ ] Rack items Reviewed

## RACK LOGIC

**Objective**  
Apply same-data logic where load and offload racks are the same physical area.

**Checklist**

- [ ] Load rack location Verified
- [ ] Offload rack location Verified
- [ ] Same physical area Confirmed
- [ ] Shared monthly data Used
- [ ] Shared weekly data Used
- [ ] Duplicate static test Avoided → Same bonding cable

---

## CALIBRATIONS

**Objective**  
Verify calibration items are recorded and attributable.

**Checklist**

- [ ] DP gauge full-range check Complete
- [ ] DP gauge travel Verified → Full down / full up
- [ ] DP interference None
- [ ] Meter calibration ownership Noted → Dwight typically handles
- [ ] Calibration entry Logged

### DECISION: Did any calibration fail or remain unsupported?

**YES**

- Escalate to supervisor / maintenance
- RECORD: Log discrepancy
- Hold affected item open

**NO**

- Submit completed dashboard item

---

## CLOSEOUT

**Objective**  
Leave the dashboard current and traceable.

**Checklist**

- [ ] Completed items Submitted
- [ ] N/A items Marked
- [ ] RECORD: Outstanding issues Logged
- [ ] Dashboard status Verified
